Smart Nutrition Platform

Kalo — Aplicație Multi-service Inteligentă

Kalo este o platformă completă de nutriție, post (fasting) și planificare a rețetelor bazată pe microservicii. Aceasta integrează un API robust dezvoltat în .NET Core pentru logica de business principală și gestiunea utilizatorilor, împreună cu un serviciu în Python (FastAPI) dedicat proceselor de Machine Learning și integrărilor AI.

Aplicația integrează de asemenea suport pentru tehnologia NFC (Near Field Communication) pentru scanarea și logarea rapidă a alimentelor în cămară.
