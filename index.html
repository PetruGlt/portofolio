Kalo — Aplicație Multi-service Inteligentă

Kalo este o platformă completă destinată managementului nutrițional bazată pe microservicii. Aceasta integrează un API robust dezvoltat în .NET Core pentru identificarea si gestionarea utilizatorilor, împreună cu un serviciu implementat în Python (FastAPI) ce cuprinde logica de bussiness,Machine Learning și integrările AI.

Aplicația integrează de asemenea suport pentru tehnologia NFC (Near Field Communication) pentru scanarea și logarea rapidă a alimentelor în cămară de către alți utilizatori.
